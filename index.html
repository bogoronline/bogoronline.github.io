Pengantar Python

Python populer atau tidak populer

Pernah dengar python? pasti mengacu kepada mahkluk hidup yang merayap dan besar yaitu ular piton atau biasa dikenal dengan ular sanca dalam bahasa sehari-hari.

Ular piton menyeramkan dan menakutkan sehingga menjadikannya populer, bagi penyuka ular, menyeramkan dan menakutkan menjadi tidak populer.

Python adalah bahasa pemrograman komputer, menjadi populer bagi pihak yang berkecimpung di bidang pengembangan aplikasi komputer, bukan ular yang merayap dan besar yang juga menyeramkan dan menakutkan.

Python alat bantu pengembangan yang komprehensif

Mengutip dari halaman resmi python (https://www.python.org), python adalah bahasa pemrograman tingkat tinggi berorientasi objek yang ditafsirkan dengan semantik dinamis. Struktur data bawaan tingkat tinggi yang dipadukan dengan pengetikan dinamis dan pengikatan dinamis membuatnya sangat menarik untuk Pengembangan Aplikasi Cepat (RPA), serta untuk digunakan sebagai bahasa skrip atau bahasa perekat untuk menghubungkan komponen-komponen yang ada. Sintaks Python yang sederhana dan mudah dipelajari menekankan keterbacaan dan oleh karena itu mengurangi biaya pemeliharaan program. Python mendukung modul dan paket, yang mendorong modularitas program dan penggunaan kembali kode. Interpreter Python dan pustaka standar yang luas tersedia dalam bentuk sumber atau biner tanpa biaya untuk semua platform utama, dan dapat didistribusikan secara bebas. Bingung juga memahaminya, sulit dicerna. Dengan kutipan diatas, cukup meyakinkan apabila python alat bantu pengembangan yang komprehensif, semuanya bisa pokoknya.

Python merupakan bahasa pemrograman yang menjadi alat bantu pengembangan ilmu data (data science), kecerdasan buatan (Artificial Intellegence), pembelajaran mesin (machine learning) dan pengembangan situs web (website development). Banyak hal yang bisa dilakukan python untuk pengembangan, blog ini fokus kepada pengembangan situs web dengan bahasa pemrograman python.

Python selalu dikaitkan dengan django

Berawal dari programmer Lawrence Journal World, yaitu Adrian Holovaty dan Simon Willison yang mulai membangun situs web dengan menggunakan bahasa pemrograman python di tahun 2003 yang berfokus membangun situs web berita online menjadi awal tumbuh dan berkembangnya kerangka kerja (framework) yang disebut django. Django dirilis secara resmi di tahun 2005 sebagai proyek gratis (open source) dan terus dimaintenance hingga saat ini dengan meningkatnya dukungan terhadap django. Pengembangan situs web dengan python memiliki keterkaitan yang kuat dengan django, bagi yang ingin menjadi programmer situs web akan menjadi pilihan terbaik untuk memulainya dan stay tune di blog ini. Dengan django framework, pembuatan dan pengembangan situs web menjadi lebih efektif, efisien dan optimasi SEO.

Python kompeten untuk membangun situs web dengan django framework

Python dan django framework merupakan alat bantu pengembangan situs web yang kompeten, salah satu contoh situs web yang menggunakan django adalah Youtube yang awalnya menggunakan bahasa pemrograman PHP. Pengembangan situs web merupakan kegiatan praktis yang terbukti secara nyata dan akan menjadi lebih baik apabila programmer situs web dapat merasakan pengalaman secara langsung. Stay tune di blog ini, semua tentang django framework.

Pengantar Django

Situs web django framework

Bagaimana mengetahui situs web yang disusun menggunakan django dengan mengakses situs web tersebut? Situs web django ditampilkan dengan ekstensi HTML, akan sulit untuk mengetahui apakah situs web dengan django atau tidak hanya dengan mengaksesnya, contoh youtube dengan ekstensi HTML. Situs web django disusun dengan menggunakan bahasa pemrograman python sehingga file pengembangan akan memiliki ekstensi py, dikarenakan HTML merupakan bahasa markup yang populer dan luas, sampai saat ini HTML menjadi pilihan utama untuk menampilkan situs web. Memulai pengembangan situs web django akan menjadi pengalaman yang berkesan dengan kecepatan serta ketepatan pengolahan data, stay tune di blog ini, semua tentang django framework.
